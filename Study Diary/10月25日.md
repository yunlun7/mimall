# 遇到的问题

1、在运行项目的时候，控制台提示：**Syntax Error: Unexpected token (25:4)**

```js

  23 |   },
  24 |   computed: {
> 25 |     ...mapState(['username'])
     |     ^
  26 |   }
  27 | }
  28 |
```

原因：webpack无法解析 ...（三点运算符），它是ES6的新特性。昨天可能是前面配置Babel的时候，将.babelrc文件删除了的原因。

解决方法：

重新配置.babelrc文件，如下：

```json
{
  "presets": [
    "stage-2"
  ]
}
```
